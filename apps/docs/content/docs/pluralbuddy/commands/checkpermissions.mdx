---
title: pb;checkpermissions
description: Check permissions of a user in the channel.
---

This command will test the permissions of the user in the currently specified channel.

## Usage

```
pb;checkpermissions
```

PluralBuddy checks the following permissions:

- Manage Messages
- Send Messages
- View Channel
- Manage Nicknames
  - Change Nickname (User)
- Manage Webhooks
