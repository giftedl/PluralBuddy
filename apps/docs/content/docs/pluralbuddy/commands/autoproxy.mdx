---
title: pb;autoproxy
description: Autoproxy command
---

This command group enables specific auto-proxy modes.

## Sub-commands

_All arguments in `<value>` format are required. All arguments in `[value]` are optional. Attachments can be uploaded directly and privately using Discord application commands as opposed to uploading directly to the command message used._

- `pb;autoproxy alter <alter>` - Proxy exclusively with an alter.
- `pb;autoproxy latch [starting alter]` - Proxy with the last alter used, and optionally start with an alter.
- `pb;autoproxy off` - Disable auto-proxy
