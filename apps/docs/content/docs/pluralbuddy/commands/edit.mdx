---
title: pb;edit
description: Edit a message with new contents.
---

This command deletes the previously sent message.

## Usage
_All arguments in `<value>` format are required. All arguments in `[value]` are optional. Attachments can be uploaded directly and privately using Discord application commands as opposed to uploading directly to the command message used._

```
pb;edit <new message contents>
```

<Callout type="warning">
    Only messages sent by your system can be edited.
</Callout>

You can reply to a message to specifically target the replied message to be edited.
