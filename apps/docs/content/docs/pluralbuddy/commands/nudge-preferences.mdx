---
title: pb;nudge-preferences
description: Configure settings for nudging
---

This command allows users to change the settings for users being nudged.

## Usage

```
pb;nudge-preferences
```

The command has numerous capabilities:

- Toggling nudging will disable nudging permanently until it is enabled again by using this command again.
- Users can be blocked from being nudged permanently.
- PluralBuddy can export your nudge blacklist.
