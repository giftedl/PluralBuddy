## Get Started with PluralBuddy

To get started using PluralBuddy, try running some commands below:
1. `pbc;setup` - Creates a system, or allows for importing from PluralKit
2. `pbc;system config` - Opens graphical configuration for your system
3. `pbc;edit-alter config <alter-name>` - Opens a graphical configuration for your alter
4. [Find more!](<https://gftl.fyi/docs>)
